# ⚙️ Four Cylinder Engine – SolidWorks Assembly

A detailed 3D assembly of a **Four Cylinder Engine** designed using **SolidWorks**. This project demonstrates mechanical assembly design, crankshaft-piston synchronization, and realistic engine mechanism visualization.

## ⚙️ Features Used

| Feature                      | Purpose                                              |
| ---------------------------- | ---------------------------------------------------- |
| `Assembly Mates`             | Defines relationships between moving components      |
| `Crankshaft Mechanism`       | Converts rotational motion into reciprocating motion |
| `Connecting Rod Assembly`    | Transfers motion between crankshaft and pistons      |
| `Rotational Constraints`     | Controls synchronized shaft movement                 |
| `Mechanical Synchronization` | Ensures proper piston timing                         |
| `Engine Assembly Design`     | Demonstrates multi-component mechanical systems      |
