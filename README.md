# ⚙️ Four Cylinder Engine – SolidWorks Assembly

A detailed 3D assembly and motion simulation of a Four Cylinder Engine designed using SolidWorks.  
This project demonstrates mechanical assembly design, crankshaft-piston synchronization, and motion simulation of an engine mechanism.
